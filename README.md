# ОПИСАНИЕ ПРОЕКТА:
Это консольная версия игры "Змейка" на Python. В ней реализованы все основные механики игры: передвижение; рост за счет поедания яблока; столкновение змейки со стеной; столкновение змейки с самой собой; ведение счета (сколько яблок было съедено по окончанию игры).

# ТЕХНОЛОГИЧЕСКИЙ СТЕК:
В качестве технологического стека был использован Python 3.x и его стандартные библиотеки.

# ОСОБЕННОСТИ РЕАЛИЗАЦИИ:
Здесь стоит отметить, что для реализации передвижения базовой змейки (голова, тело, хвост; X O O) я использовал не совсем стандартный способ. С помощью вычислений я нашел различия для каждой формы змейки и классифицировал её.
То есть для каждой формы змейки существует своя формула, что реализовано и предствлена в функции define_form(). Поэтому, когда вы вводите направление змейки, программа учитывает форму змейки на поле.
Больше информации ищите в docsrings, в самом проекте.

# УПРАВЛЕНИЕ:
w - движение вперед
d - движение вправо
a - движение влево
s - движение вниз
Quit - для выхода из игры

# СТРУКТУРА КОДА:
board() - Выводит текущее состояние поля.
clear() - Очищает поле.
start_snake() - Записывает стартовые координаты змейки для первичного вывода.
spawn_apple() - Выводит яблоко на поле.
apple_is_exists() - Проверяет, находится ли яблоко на поле.
head_coordinates_of_snake() - Возвращает координаты головы.
body_coordinates_of_snake() - Возвращает координаты тела и хвоста.
apple_coordinates_on_board() - Возвращает коордианты яблока.
define_form(head, body) - Определяет, в каком положении сейчас находится змейка на поле и возвращает название положения (строковый литерал first, second и т.д).
current_position_of_snake_and_moveing(head, body, mhc = 0, mhr = 0, phc = 0, phr = 0,
                              mb1c = 0, mb1r = 0, pb1c = 0, pb1r = 0,
                              mb2c = 0, mb2r = 0, pb2c = 0, pb2r = 0) - Двигает змейку по полю, определяет момент столкновения, увеличивает змейку от стандартной(X O O), считает полученные очки, проверяет, чтобы яблоко всегда было на поле.
eating(head, apple, mhc = 0, mhr = 0, phc = 0, phr = 0) - Функция возвращает стороковый литерал, если при то или ином движении змейки координаты головы будут равны координатам яблока.
